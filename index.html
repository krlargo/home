Overview

I'm a Software Engineer from the Bay Area. I recently graduated from UC Davis (2017) where I received my B.S. in Computer Science. While I'm open to most software engineering positions in the industry, I'm especially interested in mobile development. I've spent nearly all of my time this past summer developing iOS apps (you can check out my work below). Ideally, I'd like to land a position as a Junior iOS Software Engineer at a startup with a focus on mobile development.

I've been developing iOS apps since 2014 when Swift first released, and I've been following along with each Swift update since. I've also got some experience programming in Objective-C and React-Native. In particular, I really enjoy developing and designing the user interfaces of apps. My strongest skills regarding iOS development include using Inteface Builder, Storyboards, AutoLayout, CocoaTouch and UIKit. I also have experience using AVFoundation, Core Data, Google AdMob, SpriteKit, StoreKit (in no particular order).

You can contact me at krlargo@ucdavis.edu.

Projects

QuickSplit: A bill-splitting iOS app that utilizes a spreadsheet interface to show which users are sharing which items. Calculates in realtime, allows for varying split-portion sizes, and supports individualized tip rates.
